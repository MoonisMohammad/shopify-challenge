Installing ruby on rails on windows
1.	 Steps on a windows computer
2.	Install git Bash from this link https://gitforwindows.org/
3.	Before you run the program check to make sure that your system has the proper prerequisites installed. These include:
•	Ruby(3.0.3)
•	SQLite
•	Node.js (16.13.2 LTS)
•	Yarn (1.22.17)
4.	Install NodeJS from https://nodejs.org/ choose the one recommended for most users
5.	Open command prompt type command to install yarn: 
npm install --global yarn
6.	Install rails installer from https://web.archive.org/web/20210119202705/http://www.railsinstaller.org/en
choose windows ruby 2.3 
7.	Download and install ruby 3.0.3 with devkit from https://rubyinstaller.org/downloads/
Press enter for every message that pops up in the command prompt
8.	Open git Bash type command:
gem install rails
it will not prompt anything for some time wait for some time for the download to start
Running the program
1.	Unzip Inventory Folder
2.	Save Inventory Folder in your computer
3.	cd into inventory folder in your computer from git Bash
4.	run the command rails s in git Bash
